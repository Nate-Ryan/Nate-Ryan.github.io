About This Project

This project represents my first solo endeavor using Blender. The scene is a coffee shop render, based on a goal outlined in Polygon Runway's Tutorial. While the initial project concept was derived from the tutorial, I took full creative control of the materials, lighting, and camera positioning to make this scene my own.

One of the most challenging aspects of this project was fine-tuning the lighting to achieve the desired atmosphere. I experimented with various lighting positions, intensities, and color tones, enhancing my ability to manipulate Blender's lighting and shading tools.

This project was an invaluable learning experience, particularly in mastering how to create realistic textures and environments. It served as a solid foundation for future Blender projects and significantly boosted my confidence in 3D modeling.
