About This Project

This still image render was my first Blender project that I made independently. It was inspired by Polygon Runway's Tutorial

Only the project goal was taken from the tutorial. Feeling confident already with modeling, the big challenge of this project was the creation of materials, lighting and camera angle. I was definetly challenged by expirementing with lighting positions, strengths, and hues.

After much experimentation I was able to learn a lot of tools used within Blender for controling lighting and colors.This project was a great stepping stone into my Blender journey!
